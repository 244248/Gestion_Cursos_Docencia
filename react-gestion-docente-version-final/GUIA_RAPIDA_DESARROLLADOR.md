```

```
✅ Información Básica

   └─ Slogan: Tu lema

✅ Marca e Imágenes  
   └─ Logo URL: https://tu-logo.png

```
1. Guarda los cambios

3. ¡Listo! Tu institución personalizada
```

## 🔧 Comandos Útiles

### Iniciar el Sistema
```bash
npm start
```

### Acceder al Modo Desarrollador
```
http://localhost:3000/developer
```

### Exportar Configuración (Respaldo)
```
1. Ir a /developer
2. Clic en "Exportar Configuración"
3. Guardar el archivo JSON
```

### Importar Configuración
```
1. Ir a /developer
2. Clic en "Importar Configuración"
3. Seleccionar archivo JSON
4. Guardar cambios
```

---

## ⚠️ Importante

### ✅ Hacer Siempre
- Exportar configuración antes de cambios grandes
- Probar cambios en vista previa
- Usar URLs públicas para imágenes
- Guardar después de cada sección

### ❌ Evitar
- URLs de imágenes privadas o locales
- Colores sin formato hexadecimal (#RRGGBB)
- Cerrar sin guardar
- Cambiar todo sin probar

---

## 🆘 Solución Rápida de Problemas

| Problema | Solución |
|----------|----------|
| No se ven los cambios | Recarga con Ctrl+F5 |
| Logo no aparece | Verifica que la URL sea pública |
| Colores no cambian | Verifica formato hexadecimal |
| No puedo entrar | Contraseña: `lasalle2024dev` |
| Quiero volver atrás | Clic en "Restablecer a La Salle" |
